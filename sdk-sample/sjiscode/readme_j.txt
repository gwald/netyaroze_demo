main:	sjis 漢字コードのビューアプログラム

【起動方法】

	C:\> make		コンパイル
	C:\> siocons		siocons の起動
	[F3] auto		データおよびプログラムのダウンロード
	>> go			プログラムの実行


【操作方法】

	↑ボタン:	上方向スクロール	
	↓ボタン:	下方向スクロール	
	L1 ボタン:	文字色 +
	L2 ボタン:	文字色 -
	R1 ボタン:	背景色 +
	R2 ボタン:	背景色 -

	Key を押し続けることにより、標準速、倍速、4 倍速と
	3 段階にスピードが変化します。

【説明】

	PlayStation に内蔵されている漢字データを 4bit のテクスチャと
	して四角形のスプライトに張り付けるサンプルです。
	内蔵フォントの Shift-JIS Code を一覧することができます。特に、
	外字フォントコードの参照にご利用ください。
